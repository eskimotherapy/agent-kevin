⌨️ Keyboard shortcuts

Editing a line in the terminal

⌘ Cmd ← / ⌘ Cmd →Jump to the start / end of the line
⌥ Option ← / ⌥ Option →Move one word at a time
⌃ Ctrl WDelete the word before the cursor
⌃ Ctrl KDelete from the cursor to the end of the line
⌃ Ctrl UClear the whole line

Working with Claude

EscStop Claude mid-reply
Esc EscGo back and edit an earlier message
⇧ Shift TabSwitch modes (plan / auto-accept edits)
⌃ Ctrl CCancel what you're typing, or quit
↑Scroll back through your past messages
⇧ Shift + dragSelect / highlight text to copy (hold Shift, then drag)
/ · @ · !Slash command · mention a file · run a shell command
Press ? anytime to open this, Esc to close.
